## 🎯 학습 방식

- 🎬 **인강**: 한 입 크기로 잘라먹는 리액트 (이정환)
- 🧑‍🏫 **멘토 세션**: "왜"를 파고들며 손코딩 + 검수
- 📘 **책**: 모던 자바스크립트 Deep Dive (사전용, 막힐 때 찾기)

### 공부 루틴
```
강의 듣기 → 손코딩(따라치기) → 직접 짜보기 → 노트 정리 → 질문 검수
```

### 원칙
- 보기만 ❌ → 직접 손으로 ✅
- "왜 이렇게 되는지" 이해하고 넘어가기
- 한 번에 하나씩 (압도 방지)
- 헷갈렸던 질문은 노트에 남겨 복습
